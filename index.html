Data Sources (v1 → v2)

Simulation Logic (v1)

Synthetic data uses realistic network modeling so you can control scenarios and confidently demo system behavior.

  • latency = baseDistanceLatency + congestionFactor + randomJitter
  • packetLoss = congestion > threshold ? random(1–5%) : <1%
  • jitter = variance(latency over 60s)
